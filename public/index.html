Northcoders News Documentation

About

This webpage provides documentation on the Northcoders News API.

  • Topics
  • Articles
  • Users

GET /api/topics

This endpoint will list the slug (URL-friendly identifier) and description of each topic.

Responds with an object with key of topics, which has value containing an array of topic objects.

GET /api/articles

This endpoint will list articles with its title, author, id, topic, date, votes, image URL and comment count. They are listed from most recent.

Responds with an object with key of articles, which has value containing an array of article objects.

GET /api/users

This endpoint will list all users of the API with their usernames, name and avatar URL.

Responds with an object with key of users, which has value containing an array of user objects.

GET /api/articles/:article_id

This endpoint will respond with an article matching the requested article ID.

Responds with an object with key of article, which has value containing an article object.

GET /api/articles/:article_id/comments

This endpoint will respond with the comments of an article matching the requested article ID.

Responds with an object with key of comments, which has value containing an array comment objects.
